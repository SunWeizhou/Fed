# 3. 方法 (Methodology)

## 3.1 框架概述 (Framework Overview)

本文采用两阶段方法定义 **ACT-FedViM**。第一阶段训练标准的联邦 ViM 基线（Fed-ViM），在联邦训练过程中聚合全局充分统计量并得到原始 ViM 子空间；第二阶段在不重新训练 backbone 的前提下，对同一 checkpoint 执行 **ACT 后处理**，以自适应方式重新选择更紧凑的子空间维度，并据此重新校准 ViM 评分。换言之，本文中的 **ACT-FedViM 本质上是原始 Fed-ViM 的 ACT 后处理增强版本**，而不是一个单独的端到端训练框架。

因此，整体流程包含三个步骤：

1. **训练原始 Fed-ViM 并保存联邦统计量**：客户端完成本地训练，并上传一阶、二阶统计量；服务端据此重构全局协方差矩阵与原始 ViM 子空间。
2. **ACT 后处理式自适应子空间选择**：在训练完成后，基于保存的全局统计量，对相关矩阵特征值进行 ACT 修正，并自动确定新的子空间维度 $k_{\text{opt}}$。
3. **ACT 子空间下的 ViM 重新校准与 OOD 评分**：在新的子空间上重新估计或复用 $\alpha$，并通过 ViM 联合得分完成 OOD 检测。

这一表述与当前代码实现保持一致：训练阶段得到的是 Original Fed-ViM，ACT-FedViM 则是在训练后对同一模型进行的统计后处理。

## 3.2 原始 Fed-ViM 的联邦统计量聚合 (Federated Statistics Aggregation for Original Fed-ViM)

在标准联邦学习设定下，设有 $N$ 个客户端，每个客户端 $i$ 拥有本地数据集 $\mathcal{D}_i = \{(x_j^{(i)}, y_j^{(i)})\}_{j=1}^{n_i}$，特征提取器记为 $f_\theta(\cdot) \in \mathbb{R}^D$。

在本地训练的最后一个 epoch 中，客户端在完成参数更新的同时累计特征的充分统计量：

$$s_i^{(1)} = \sum_{x \in \mathcal{D}_i} f_\theta(x), \quad s_i^{(2)} = \sum_{x \in \mathcal{D}_i} f_\theta(x) f_\theta(x)^T, \quad c_i = |\mathcal{D}_i|$$

客户端上传 $\{s_i^{(1)}, s_i^{(2)}, c_i\}$ 与模型参数，服务端据此重构全局特征均值与协方差矩阵：

$$\mu_{\text{global}} = \frac{\sum_{i=1}^N s_i^{(1)}}{\sum_{i=1}^N c_i}$$

$$\Sigma_{\text{global}} = \frac{\sum_{i=1}^N s_i^{(2)}}{\sum_{i=1}^N c_i} - \mu_{\text{global}}\mu_{\text{global}}^T$$

在原始 Fed-ViM 中，服务端基于 $\Sigma_{\text{global}}$ 做标准 PCA。需要强调的是，原始 ViM 论文中的主子空间维度 $k$ 主要依赖人为设定或经验范围，并不存在统一的闭式最优规则。本文在联邦化实现里进一步采用固定方差贡献率准则（实现中默认为 95%）作为一个动态 heuristic 来确定原始子空间维度 $k_{\text{orig}}$，得到原始投影矩阵 $P_{\text{orig}}$。这一步对应本文的 `Original ViM` 基线。

## 3.3 ACT 后处理式自适应子空间选择 (Post-hoc Adaptive Subspace Selection via ACT)

原始 ViM 的一个关键超参数是主子空间维度 $k$。在原始论文中，这一维度主要通过固定值或经验范围来设定，并未给出统一的闭式最优解。本文的联邦化 ViM 基线进一步使用累计方差贡献率（默认 95%）作为动态 heuristic 来选取 $k$。这一做法相比固定 `k` 更灵活，但在高维特征空间中仍然面临经典 PCA 的统计局限：当 $p/n \to \gamma > 0$ 时，样本协方差矩阵的特征谱会受到噪声污染，导致谱尾膨胀，进而使主子空间规模选择偏大，甚至使主特征方向的估计也变得不稳定。为此，本文在训练完成后引入 **ACT (Adjusted Correlation Thresholding)** 作为后处理模块，对原始 Fed-ViM 的子空间维度进行自适应修正。

具体地，给定由联邦统计量重构得到的全局协方差矩阵 $\Sigma_{\text{global}}$，我们先将其转化为相关矩阵：

$$R = D^{-1/2} \Sigma_{\text{global}} D^{-1/2}$$

其中 $D$ 为 $\Sigma_{\text{global}}$ 对角元素构成的尺度矩阵。对相关矩阵做特征值分解，记降序特征值为 $\lambda_1 \ge \lambda_2 \ge \cdots \ge \lambda_p$。根据 Marchenko-Pastur 理论，在纯噪声假设下的阈值为：

$$s = 1 + \sqrt{\frac{p}{n - 1}}$$

其中 $p$ 为特征维度，$n$ 为总训练样本数。

随后，ACT 通过修正的 Stieltjes 变换对样本特征值进行偏差修正。对第 $j$ 个特征值，定义：

$$m_{nj}(z) = \frac{1}{p - j} \left[ \sum_{l=j+1}^p \frac{1}{\lambda_l - z} + \frac{4}{3\lambda_j + \lambda_{j+1} - 4z} \right]$$

$$\underline{m}_{nj}(z) = -\frac{1 - \rho_j}{z} + \rho_j m_{nj}(z), \quad \rho_j = \frac{p - j}{n - 1}$$

进而得到修正特征值估计：

$$\lambda_j^C = -\frac{1}{\underline{m}_{nj}(\lambda_j)}$$

最终，ACT 通过以下准则自适应确定新的有效维度：

$$k_{\text{opt}} = \max \{ j : \lambda_j^C > s \}$$

需要强调的是，当前实现中 **ACT 的作用是后处理地确定新的子空间维度 $k_{\text{opt}}$**。在得到该维度后，我们仍然在全局协方差矩阵上执行 PCA，并取前 $k_{\text{opt}}$ 个方向构造新的投影矩阵 $P_{\text{ACT}}$。因此，ACT-FedViM 的本质是“ACT-guided subspace refinement for ViM”，即用 ACT 替代本文联邦 ViM 基线中的经验性固定方差阈值 heuristic，为 ViM 提供更有统计依据的子空间规模选择。更具体地说，ACT 不是直接学习新的特征方向，而是通过随机矩阵理论识别并截断噪声主导的谱尾，从而让后续 PCA 得到的主子空间方向估计更加可靠。

## 3.4 ACT-FedViM 的 ViM 重校准与 OOD 评分 (ViM Recalibration and OOD Scoring after ACT)

在得到 ACT 后处理子空间 $P_{\text{ACT}}$ 后，我们对 ViM 打分进行重新校准。对于输入样本 $x$，记其特征为 $z = f_\theta(x)$，分类器输出 logits 为 $v$，则几何残差定义为：

$$\text{Residual}(x) = \| (I - P_{\text{ACT}} P_{\text{ACT}}^T)(z - \mu_{\text{global}}) \|_2$$

能量项定义为：

$$\text{Energy}(x) = \log \sum_{c=1}^C \exp(v_c)$$

在当前正式实验口径中，原始 Fed-ViM 与 ACT-FedViM 均统一采用基于 ID 训练特征的经验校准策略。具体而言，给定当前子空间投影矩阵 $P$ 与全局均值 $\mu_{\text{global}}$，我们在 ID 训练特征上计算：

$$\bar{R}_{\text{train}} = \frac{1}{N}\sum_{i=1}^N \left\|(I - PP^T)(z_i-\mu_{\text{global}})\right\|_2$$

$$\bar{E}_{\text{train}} = \frac{1}{N}\sum_{i=1}^N \log \sum_{c=1}^C \exp(v_c(z_i))$$

$$\alpha = \frac{|\bar{E}_{\text{train}}|}{\bar{R}_{\text{train}} + \epsilon}$$

因此，无论是原始 ViM 子空间还是 ACT 后处理子空间，本文都在对应的投影矩阵下重新经验估计 $\alpha$，从而保证两种方法在评分口径上完全一致，避免性能增益混入不同校准策略带来的偏差。

最终，ACT-FedViM 使用与原始 ViM 相同的联合评分形式：

$$\text{Score}(x) = \text{Energy}(x) - \alpha \cdot \text{Residual}(x)$$

在该定义下，ID 样本通常具有更高的能量和更小的残差，因此得分更高；而 OOD 样本往往表现为低能量与高残差，因此得分更低。实验中通过比较 ID 与 OOD 样本得分分布，并计算 AUROC、AUPR 与 FPR95 等指标，评估 ACT 后处理对原始 ViM 的增益。

## 3.5 方法口径与实验对照关系

基于上述定义，本文实验中的方法关系如下：

- **Original ViM / Fed-ViM**：联邦训练完成后，使用固定方差贡献率 PCA 这一经验性 heuristic 构造的原始 ViM 子空间。
- **ACT-FedViM (Ours)**：在相同 checkpoint 和相同联邦统计量基础上，使用 ACT 后处理自适应选择新的子空间维度，并重新执行 ViM 评分。
- **MSP / Energy**：作为联邦设定下可直接实现的独立 OOD baseline，与原始 ViM 和 ACT-FedViM 在相同 backbone 上做公平对比。`Mahalanobis` 需要服务器访问带标签的训练特征，不符合本文的联邦隐私假设，因此不纳入正式对比。

因此，本文方法的核心贡献不在于重新训练一个新的分类器，而在于：**在无需重新训练 backbone 的前提下，通过 ACT 后处理提升联邦 ViM 的子空间选择质量与 OOD 检测表现。**
