拟定论文题目：**FedViM：面向多中心海洋浮游生物监测的联邦分布外检测方法研究**

## 1. 选题依据及研究内容

### 1.1 研究背景与意义

浮游生物是海洋生态系统中的重要组成部分，其种类、数量和分布情况与海洋生态环境变化密切相关[1]。随着显微成像设备和深度学习技术的发展，浮游生物监测已经逐步从人工镜检转向自动化图像识别[2-4]。这使得监测效率得到提高，但也带来了新的问题：一是不同单位往往各自保存本地数据，原始图像不方便直接集中；二是真实监测中不仅有已知类别，还会出现未见类别和各种非目标样本，单纯做分类并不能满足实际需求。

在多中心海洋监测场景下，原始图像及其采样元信息可能涉及采样位置、设备布放和任务背景等内容，跨中心直接汇聚数据往往会受到隐私、管理和传输成本等方面的限制。联邦学习通过“数据不出本地”的方式，为多中心数据协作提供了思路[5-7]。但是，仅有联邦分类模型还不够，系统还需要具备识别异常样本和未知样本的能力。

另一方面，海洋浮游生物监测具有明显的开放世界特征。实际应用中，模型除了面对训练集中出现过的浮游生物类别，还可能遇到未知种类、鱼卵、鱼尾、气泡和颗粒杂波等样本[8-10]。如果系统把这些样本错误地识别成已知类别，就会影响监测结果的可靠性。因此，研究联邦场景下的分布外检测方法，既有实际意义，也有一定的理论价值。

基于以上背景，本文拟研究的问题是：**在不共享原始图像和单样本特征的前提下，如何为联邦学习得到的浮游生物分类模型构建后处理式 OOD 检测方法。**

### 1.2 国内外研究现状

目前与本课题相关的研究主要包括以下几个方面。

第一，海洋浮游生物图像识别方面，已有研究已经从传统人工特征方法逐步发展到深度学习方法，并开始面向高通量监测、边缘部署和现场应用展开探索[2-4]。但已有工作也指出，浮游生物识别容易受到数据分布变化、未见类别和非目标干扰样本的影响，因此这一任务并不只是普通的封闭集分类问题[8-10]。

第二，联邦学习方面，FedAvg 及其后续研究为多中心数据协作提供了基础框架[5-7]。现有研究主要集中在模型训练、通信效率和非独立同分布数据等问题上。近几年也开始有工作关注联邦场景下的 OOD 检测[11]，但不少方法与训练阶段的额外模块结合得比较紧，整体实现较复杂。

第三，后处理式 OOD 检测方面，`MSP`[12] 和 `Energy`[13] 是较经典的输出空间基线方法，部署比较简单；`ViM`[14] 进一步利用特征空间结构，在多个视觉任务中表现较好。已有研究表明，ViM 在浮游生物 OOD 任务上也具有较好的表现[10]。不过，ViM 依赖全局特征均值与协方差，而它的主子空间维度 $k$ 一般依赖经验设定，这使得它在联邦场景下的实现和改造仍有研究空间。

此外，高维统计中的 ACT 方法[15] 为特征空间有效维度选择提供了一种数据驱动思路。它是否可以作为 ViM 的扩展模块，用于联邦场景下的主子空间选维，也是一个值得研究的问题。

总体来看，现有研究分别在浮游生物识别、联邦学习、后处理 OOD 检测和高维统计选维等方面提供了基础，但缺少一种面向多中心海洋浮游生物监测任务、能够在不共享原始图像条件下复用联邦分类模型的后处理 OOD 方法。

### 1.3 初步设想及拟解决的问题

根据前期阅读文献和已有实验准备，本文的初步设想是：以 ViM 为基础，将其所需的全局统计量改写为客户端一阶、二阶特征统计量的联邦聚合过程，构造联邦后处理 OOD 方法 `FedViM`；在此基础上，再进一步考察能否引入 ACT 进行主子空间维度的自适应选择，形成 `ACT-FedViM`。

本文拟重点解决以下几个问题：

1. 在不共享原始图像和单样本特征的条件下，如何得到 ViM 所需的全局特征均值与协方差。
2. 在联邦后处理场景中，如何构造适合当前任务的 OOD 评分方式。
3. 在联邦 ViM 框架下，ACT 自适应选维是否能够减少 fixed-k 经验设定带来的问题。
4. 在海洋浮游生物任务上，联邦 ViM 路线与 `MSP`、`Energy` 等基线相比是否具有应用价值。

本文的思路不是把 `ACT-FedViM` 单独作为唯一中心，而是以 `FedViM` 为主方法，把 `ACT-FedViM` 作为扩展模块进行研究和比较。

### 1.4 研究目标与预期成果

本文的研究目标是构建并评估一套面向多中心海洋浮游生物监测场景的联邦后处理 OOD 检测方法。具体目标包括：

1. 设计并实现 `FedViM`，完成全局统计量的联邦重构。
2. 研究 `ACT-FedViM` 的自适应选维方式，并分析其作用。
3. 在统一实验设置下，对多种 CNN 模型进行测试，并与 `MSP`、`Energy` 等基线进行比较。

预期成果主要包括：

1. 完成毕业论文写作，形成与题目相一致的研究主线。
2. 完成 `FedViM` 与 `ACT-FedViM` 的实现与实验评估。
3. 形成实验结果表格、图示以及相应分析。

### 1.5 拟突破点

本文拟在以下几个方面做出尝试：

1. 将 ViM 从集中式后处理方法推广到联邦场景，形成可实现的 `FedViM`。
2. 在联邦统计量可聚合的前提下，引入 ACT 作为扩展模块，为主子空间维度选择提供更有依据的方式。
3. 结合海洋浮游生物监测这一具体任务，对联邦后处理 OOD 方法开展较系统的实验分析。

## 2. 可行性论述

本课题具有较好的可行性。

首先，在数据基础方面，本文拟使用 `DYB-PlanktonNet` 数据集及其 OOD 划分开展实验[10][16]。该数据集同时包含 ID 类别和 OOD 样本，能够支撑联邦分类训练与后处理检测评估。

其次，在方法基础方面，本文采用的是后处理思路，即在已经训练完成的联邦分类模型基础上增加 OOD 检测模块，而不是重新设计完整训练框架。这样可以降低实现复杂度，也便于和已有分类模型结合。

再次，在前期准备方面，目前已经完成了一部分文献整理、数据组织和代码框架搭建工作，对联邦训练流程和后处理评估流程已有初步了解，因此课题具备继续推进的条件。

## 3. 研究方法与技术路线

本文拟采用“联邦分类模型 + 后处理 OOD 检测”的思路开展研究，基本技术路线如下：

1. 使用联邦学习方式训练浮游生物分类模型，并保存最佳模型参数。
2. 各客户端在本地 ID 训练集上提取特征，计算一阶与二阶统计量，并上传到服务器。
3. 服务器聚合各客户端统计量，重构全局特征均值与协方差。
4. 基于重构得到的统计量实现 `FedViM`，计算后处理 OOD 分数。
5. 在相同统计量基础上引入 ACT，构造 `ACT-FedViM` 并比较其效果。
6. 选取 `MSP`、`Energy` 作为基线方法，在多个 CNN 模型上进行比较。
